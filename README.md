# Casa Trigo Zero - Landing Page

Landing page institucional da **Casa Trigo Zero**, desenvolvida com HTML, CSS e JavaScript puro.

## Visão Geral

O projeto apresenta:
- Seção principal com chamada para ação de pedidos
- Cards de produtos em destaque
- Seção de diferenciais
- Rodapé com contato, Instagram, WhatsApp e localização com mapa incorporado
- Layout responsivo para desktop e mobile
- Animações de entrada com `IntersectionObserver`

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts (`Fraunces` e `Manrope`)

## Estrutura do Projeto

```txt
.
├── index.html
├── style.css
├── script.js
└── README.md
```

## Como Rodar Localmente

1. Clone este repositório:

```bash
git clone <url-do-repositorio>
```

2. Acesse a pasta:

```bash
cd <nome-da-pasta>
```

3. Abra o arquivo `index.html` no navegador.

Opcionalmente, você pode usar uma extensão como **Live Server** no VS Code.

## Customização Rápida

- Textos e seções: edite `index.html`
- Cores, espaçamentos e responsividade: edite `style.css`
- Menu mobile e animações: edite `script.js`

## Deploy

Você pode publicar facilmente em:
- GitHub Pages
- Netlify
- Vercel

Basta enviar os arquivos estáticos (`index.html`, `style.css`, `script.js` e imagens).
